20. л°ұм—… кҙҖлҰ¬

v0.6.23 м¶”к°Җ

л°ұм—… мҠӨмјҖмӨ„ кҙҖлҰ¬, мӢӨн–ү мқҙл Ҙ мЎ°нҡҢ, л°ұм—… нҢҢмқј мқёлІӨнҶ лҰ¬лҘј мӣ№ UIм—җм„ң нҶөн•© кҙҖлҰ¬н•ҳкі , л°ұм—… нҸ¬мқёнҠёлЎңл¶Җн„° лҚ°мқҙн„°лҘј ліөмӣҗн•©лӢҲлӢӨ.

20.1 к°ңмҡ”вҖӢ

мӮ¬мқҙл“ңл°”м—җм„ң мҠӨнҶ лҰ¬м§Җ > л°ұм—… кҙҖлҰ¬лҘј м„ нғқн•©лӢҲлӢӨ. 4к°ңмқҳ нғӯмңјлЎң кө¬м„ұлҗ©лӢҲлӢӨ:

  1. мҠӨмјҖмӨ„ вҖ” л“ұлЎқлҗң л°ұм—… мҠӨмјҖмӨ„ лӘ©лЎқ л°Ҹ мҰүмӢң мӢӨн–ү
  2. мқҙл Ҙ вҖ” л°ұм—… мӢӨн–ү кІ°кіј нһҲмҠӨнҶ лҰ¬
  3. мқёлІӨнҶ лҰ¬ вҖ” л””мҠӨнҒ¬м—җ мЎҙмһ¬н•ҳлҠ” л°ұм—… нҢҢмқј лӘ©лЎқ
  4. ліөмӣҗ вҖ” л°ұм—… нҸ¬мқёнҠё м„ нғқ л°Ҹ лҚ°мқҙн„° ліөмӣҗ (v0.6.28 м¶”к°Җ)

20.2 л°ұм—… мҠӨмјҖмӨ„вҖӢ

м»¬лҹјм„ӨлӘ…
мҠӨмјҖмӨ„лӘ…л°ұм—… мһ‘м—… мқҙлҰ„
Cron н‘ңнҳ„мӢқмӢӨн–ү мЈјкё° (Cron нҳ•мӢқ)
ліҙкҙҖ мқјмҲҳл°ұм—… нҢҢмқј ліҙкҙҖ кё°к°„
л§Ҳм§Җл§ү мғҒнғңмөңк·ј мӢӨн–ү кІ°кіј (м„ұкіө/мӢӨнҢЁ/мӢӨн–үмӨ‘/кұҙл„ҲлңҖ)
л§Ҳм§Җл§ү мӢӨн–үмөңк·ј мӢӨн–ү мӢңк°Ғ
мҰүмӢң мӢӨн–үмҲҳлҸҷ мӢӨн–ү лІ„нҠј

мҰүмӢң мӢӨн–үвҖӢ

мҰүмӢң мӢӨн–ү лІ„нҠјмқ„ нҒҙлҰӯн•ҳл©ҙ нҷ•мқё лҢҖнҷ”мғҒмһҗк°Җ н‘ңмӢңлҗ©лӢҲлӢӨ. мҠ№мқё мӢң н•ҙлӢ№ мҠӨмјҖмӨ„мқҳ л°ұм—…мқҙ мҰүмӢң мӢӨн–үлҗ©лӢҲлӢӨ.

20.3 мӢӨн–ү мқҙл ҘвҖӢ

м»¬лҹјм„ӨлӘ…
мҠӨмјҖмӨ„лӘ…мӢӨн–үлҗң л°ұм—… мҠӨмјҖмӨ„ мқҙлҰ„
мғҒнғңкІ°кіј (м„ұкіө/мӢӨнҢЁ/мӢӨн–үмӨ‘)
мӢңмһ‘ мӢңк°Ғл°ұм—… мӢңмһ‘ мӢңм җ
мҷ„лЈҢ мӢңк°Ғл°ұм—… мҷ„лЈҢ мӢңм җ
м¶ңл Ҙ/мҳӨлҘҳмӢӨн–ү лЎңк·ё лҳҗлҠ” мҳӨлҘҳ л©”мӢңм§Җ

20.4 л°ұм—… мқёлІӨнҶ лҰ¬вҖӢ

м»¬лҹјм„ӨлӘ…
л°ұм—… IDкі мң  мӢқлі„мһҗ
мң нҳ•л°ұм—… лҢҖмғҒ (MariaDB/OpenSearch)
нҒ¬кё°л°ұм—… нҢҢмқј нҒ¬кё°
кІҖмҰқл¬ҙкІ°м„ұ кІҖмҰқ кІ°кіј (вң“/вң—)
мғқм„ұ мӢңк°Ғл°ұм—… нҢҢмқј мғқм„ұ мӢңм җ
нҢҢмқј кІҪлЎңл””мҠӨнҒ¬ мғҒмқҳ нҢҢмқј мң„м№ҳ

20.5 лҚ°мқҙн„° ліөмӣҗвҖӢ

v0.6.28 м¶”к°Җ

ліөмӣҗ нғӯм—җм„ң л°ұм—… нҸ¬мқёнҠёлҘј м„ нғқн•ҳм—¬ лҚ°мқҙн„°лҘј ліөмӣҗн•©лӢҲлӢӨ.

ліөмӣҗ лҢҖмғҒвҖӢ

лҢҖмғҒм„ӨлӘ…
MariaDB м „мІҙ ліөмӣҗmariabackup кё°л°ҳ м „мІҙ л¬јлҰ¬ ліөмӣҗ
MariaDB нҢҢнӢ°м…ҳ ліөмӣҗнҠ№м • нҢҢнӢ°м…ҳмқҳ л…јлҰ¬ ліөмӣҗ (mysqldump кё°л°ҳ)
OpenSearch мҠӨлғ…мғ· ліөмӣҗмҠӨлғ…мғ· кё°л°ҳ мқёлҚұмҠӨ ліөмӣҗ

ліөмӣҗ м Ҳм°ЁвҖӢ

  1. ліөмӣҗ лҢҖмғҒ мң нҳ•мқ„ м„ нғқн•©лӢҲлӢӨ (MariaDB / Partition / OpenSearch).
  2. мӮ¬мҡ© к°ҖлҠҘн•ң л°ұм—… нҸ¬мқёнҠё лӘ©лЎқм—җм„ң ліөмӣҗн•  л°ұм—…мқ„ м„ нғқн•©лӢҲлӢӨ.
  3. мӢңл®¬л Ҳмқҙм…ҳ лІ„нҠјмңјлЎң мӮ¬м „ кІҖмҰқмқ„ мҲҳн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ (Partitionл§Ң м§Җмӣҗ).
  4. ліөмӣҗ мӢӨн–ү лІ„нҠјмқ„ нҒҙлҰӯн•©лӢҲлӢӨ.
  5. 1лӢЁкі„ нҷ•мқё: ліөмӣҗ лҢҖмғҒкіј л°ұм—… нҸ¬мқёнҠёлҘј нҷ•мқён•©лӢҲлӢӨ.
  6. 2лӢЁкі„ нҷ•мқё: мөңмў… кІҪкі  ("мқҙ мһ‘м—…мқҖ лҗҳлҸҢлҰҙ мҲҳ м—ҶмҠөлӢҲлӢӨ") нӣ„ мӢӨн–үн•©лӢҲлӢӨ.
  7. ліөмӣҗ мҷ„лЈҢ мӢң кІ°кіј(м„ұкіө/мӢӨнҢЁ, мҶҢмҡ” мӢңк°„)к°Җ н‘ңмӢңлҗ©лӢҲлӢӨ.
warning

ліөмӣҗмқҖ нҳ„мһ¬ лҚ°мқҙн„°лҘј лҚ®м–ҙм”ҒлӢҲлӢӨ. ліөмӣҗ м „ л°ҳл“ңмӢң нҳ„мһ¬ мғҒнғңлҘј нҷ•мқён•ҳкі , мӨ‘мҡ” лҚ°мқҙн„°к°Җ мһҲлӢӨл©ҙ лЁјм Җ лі„лҸ„ л°ұм—…мқ„ мҲҳн–үн•ҳм„ёмҡ”.

info

мӢңл®¬л Ҳмқҙм…ҳ лӘЁл“ңлҠ” нҳ„мһ¬ MariaDB нҢҢнӢ°м…ҳ ліөмӣҗм—җм„ңл§Ң м§Җмӣҗлҗ©лӢҲлӢӨ. мӢңл®¬л Ҳмқҙм…ҳ мӢң мӢӨм ң лҚ°мқҙн„°лҠ” ліҖкІҪлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

20.6 API м—”л“ңнҸ¬мқёнҠёвҖӢ

MethodPathм„ӨлӘ…
GET/api/backup/schedulesл°ұм—… мҠӨмјҖмӨ„ лӘ©лЎқ
POST/api/backup/run/{id}мҠӨмјҖмӨ„ мҰүмӢң мӢӨн–ү
GET/api/backup/historyмӢӨн–ү мқҙл Ҙ мЎ°нҡҢ
GET/api/backup/inventoryл°ұм—… нҢҢмқј мқёлІӨнҶ лҰ¬
POST/api/backup/verify/{id}л°ұм—… л¬ҙкІ°м„ұ кІҖмҰқ (SHA-256 мІҙнҒ¬м„¬, нҒ¬кё° нҷ•мқё)
POST/api/restore/mariadbMariaDB м „мІҙ ліөмӣҗ (JSON body: {full_backup_path, confirm})

POST/api/restore/opensearchOpenSearch мҠӨлғ…мғ· ліөмӣҗ (JSON body: {repository, snapshot, indices, confirm})
