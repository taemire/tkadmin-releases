17. 리포트마법사

리포트마법사는 TACHYON 솔루션의 49종 로그 테이블에 대해 드래그 앤 드롭(DnD) 위자드 방식으로 커스텀 보고서를 생성하는 기능입니다. SQL 지식 없이도 테이블·컬럼 선택, 날짜 범위·필터·정렬 조건 설정, 미리보기 및 Excel/PDF 내보내기를 수행할 수 있습니다.

17.1 접근 방법​

사이드바 도구 섹션에서 리포트마법사를 클릭합니다.

17.2 화면 구성​

리포트마법사는 5단계 위자드 방식으로 구성됩니다.

Step 0: 랜딩 페이지 (시작 화면)​

리포트마법사 진입 시 가장 먼저 표시되는 화면입니다.

항목설명
신규 리포트 생성빈 상태에서 새 보고서를 처음부터 만듭니다
템플릿 불러오기이전에 저장한 리포트 템플릿을 불러와 시작합니다
이어서 작업이전 세션에서 작업 중이던 리포트가 있으면 상단에 배너로 표시됩니다

참고: 위자드 진행 중 상단 홈 버튼(🏠)을 클릭하면 언제든 이 화면으로 돌아올 수 있습니다.

Step 1: 테이블 선택​

TACHYON 로그 테이블 카탈로그에서 보고서에 포함할 테이블을 선택합니다.

9개 카테고리:

카테고리테이블 수설명
에이전트4에이전트 이벤트, 시스템, 보안정책, 네트워크 로그
매체제어7USB, 파일, 와이파이, 블루투스, 프린터, 화면 캡처 로그
보안USB6USB 이벤트, 파일, 복사, 승인, 반출, 폐기 로그
악성코드2백신 탐지, APT 탐지 로그
네트워크 보안2IPS, 방화벽 로그
랜섬웨어2랜섬웨어 탐지, MBR 보호 로그
자체보호/서버5서버 이벤트, 자가 진단, 업데이트, 정책, 명령 로그
패치/기타2패치 관리, 자산 관리 로그
통계6일간/주간/월간 요약, 에이전트 현황, 이벤트 추이, 보안 점수
  • 카테고리 카드를 클릭하면 해당 테이블 목록이 펼쳐집니다
  • 체크박스를 클릭하여 여러 테이블을 선택할 수 있습니다

Step 2: 컬럼 선택​

선택한 테이블의 컬럼을 보고서에 추가합니다.

영역설명
컬럼 팔레트 (좌측)사용 가능한 컬럼 목록. 체크박스를 클릭하여 추가/제거
리포트 캔버스 (우측)선택된 컬럼 목록. 드래그로 순서 변경, ✕로 삭제, 에일리어스 인라인 편집
  • 각 컬럼 칩에는 컬럼명, 데이터 타입 뱃지, 설명 서브타이틀이 표시됩니다
  • 컬럼 칩에 마우스를 올리면 설명 전문이 툴팁으로 표시됩니다
  • 코드 해석이 적용되는 컬럼에는 🔄 아이콘이 표시되며, 자동으로 코드→레이블 변환이 활성화됩니다

Step 3: 조건 설정​

보고서의 날짜 범위, 필터 조건, 정렬 기준을 설정합니다.

날짜 범위​

태그를 클릭하여 빠르게 설정할 수 있습니다:

태그설명
사용안함날짜 필터 없이 전체 데이터 조회 (기본값)
최근 1주오늘로부터 7일 이전 ~ 현재
최근 15일오늘로부터 15일 이전 ~ 현재
최근 1개월오늘로부터 30일 이전 ~ 현재
직접 입력시작일시/종료일시를 직접 입력

필터 조건​

+ 필터 추가 버튼을 클릭하여 동적으로 조건을 추가합니다.

설정 항목설명
컬럼필터를 적용할 컬럼 선택
연산자=, !=, >=, <=, LIKE, BETWEEN, IN
값비교할 값 입력 (BETWEEN일 경우 시작/끝 값)

정렬​

+ 정렬 추가 버튼을 클릭하여 정렬 기준을 추가합니다.

설정 항목설명
컬럼정렬 기준 컬럼
방향ASC (오름차순) / DESC (내림차순)

Step 4: 미리보기 & 내보내기​

최종 보고서를 확인하고 다양한 형식으로 내보냅니다.

기능설명
SQL 미리보기생성된 SELECT 쿼리문 확인
미리보기 실행쿼리를 실행하여 데이터 테이블로 결과 확인
Excel 내보내기XLSX 파일 다운로드 (정보 시트 + 데이터 시트)
PDF 내보내기PDF 파일 다운로드 (A4 가로, 최대 100,000행)
템플릿 저장현재 설정을 템플릿으로 저장하여 재사용

17.3 템플릿 관리​

자주 사용하는 보고서 설정을 템플릿으로 저장하고 불러올 수 있습니다.

저장​

Step 4에서 템플릿 저장 버튼을 클릭합니다.

입력 항목설명
템플릿 이름보고서 템플릿의 이름
설명템플릿에 대한 간단한 설명 (선택)
공유 범위private (본인만) / admin (관리자 공유) / system (전체 공유)

불러오기​

랜딩 페이지(Step 0)에서 템플릿 불러오기를 클릭하면 저장된 템플릿 목록이 표시됩니다. 원하는 템플릿을 선택하면 설정이 자동 로드되어 Step 1부터 시작됩니다.

17.4 코드 자동 해석​

TACHYON은 다양한 상태값을 숫자 코드로 저장합니다 (예: 에이전트 상태 1=정상, 2=비정상). 리포트마법사는 TPU_INFO_CODE 테이블을 참조하여 이러한 코드를 자동으로 사람이 읽을 수 있는 레이블로 변환합니다.

  • 미리보기 및 내보내기 결과에 자동 적용
  • 별도 설정 불필요

17.5 보안​

정책설명
읽기 전용SELECT 쿼리만 실행 가능 (데이터 변경 불가)
테이블 제한TACHYON 로그 테이블(TPU_ISTAT_*)만 접근 가능
SQL 인젝션 방지모든 입력값은 파라미터 바인딩으로 처리
인증 필수로그인된 사용자만 접근 가능
템플릿 권한본인 생성 또는 공유 범위에 따라 접근 제어

17.6 API 엔드포인트​

Method경로설명
GET/api/querywizard/report/catalog로그 테이블 카탈로그
GET/api/querywizard/report/codes코드→레이블 매핑
POST/api/querywizard/report/preview미리보기 실행
POST/api/querywizard/report/export/excelExcel 내보내기
POST/api/querywizard/report/export/pdfPDF 내보내기
GET/api/querywizard/report/templates템플릿 목록
POST/api/querywizard/report/templates템플릿 저장
PUT/api/querywizard/report/templates/:id템플릿 수정
DELETE/api/querywizard/report/templates/:id템플릿 삭제

17.7 활용 사례​

주간 에이전트 이벤트 보고서​

  1. Step 1: 에이전트 → TPU_ISTAT_EVENT_LOG 선택
  2. Step 2: server_dt, agent_ip, event_type, event_detail 컬럼 선택
  3. Step 3: 날짜 범위 최근 1주 선택, 정렬 server_dt DESC
  4. Step 4: 미리보기 확인 후 Excel 내보내기

매체제어 월간 요약 보고서​

  1. Step 1: 매체제어 → TPU_ISTAT_MEDIA_LOG, TPU_ISTAT_MEDIA_FILE_LOG 선택
  2. Step 2: 필요 컬럼 선택
  3. Step 3: 날짜 범위 최근 1개월, 필터 event_type = USB_COPY
  4. Step 4: 미리보기 후 PDF 내보내기, 템플릿 저장
