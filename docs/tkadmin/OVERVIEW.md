# tkadmin 운영자 메뉴얼에 오신 것을 환영합니다

이 메뉴얼은 **tkadmin**(TACHYON 솔루션 웹 기반 설정 관리 및 운영 자동화 도구)의 운영자를 위한 사용 가이드입니다.

## 메뉴얼 구성

이 메뉴얼은 다음과 같은 섹션으로 구성되어 있습니다.

| 섹션 | 설명 |
|------|------|
| **서비스 소개** | tkadmin의 개요, 주요 기능, 지원 환경을 소개합니다. |
| **시작하기** | 설치, 초기 설정, 첫 로그인 과정을 안내합니다. |
| **화면별 기능 가이드** | 대시보드, 설정 편집기, 서비스 관리 등 각 화면의 기능을 상세히 설명합니다. |
| **CLI 옵션 가이드** | 명령줄 인터페이스(CLI) 옵션과 사용법을 다룹니다. |
| **FAQ 및 문제 해결** | 자주 묻는 질문과 문제 해결 방법을 정리합니다. |
